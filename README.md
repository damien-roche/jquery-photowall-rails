# Photowall::Rails

ORIGINAL
========
- https://github.com/creotiv/jquery-photowall

## Installation

Add this line to your application's Gemfile:

    gem 'jquery-photowall-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jquery-photowall-rails
